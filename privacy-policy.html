資料儲存

本 App 可能由使用者自行建立商品分類、商品名稱、條碼、有效日期、主題設定與備份資料。

上述資料預設儲存在裝置本機 IndexedDB，不會由本 App 主動上傳到 GitHub Pages 或其他伺服器。

若使用「本機檔案位置」，資料會寫入使用者選擇的本機 JSON 檔案；使用者可自行保管、備份或刪除該檔案。

相機權限

相機只用於條碼掃描。Android App 使用裝置內的 ML Kit 辨識條碼；網頁版則使用瀏覽器支援的條碼辨識能力。

本專案不會儲存相機影像，不會錄音，也不會把影像傳送到遠端服務。

若使用者拒絕相機權限，仍可手動輸入條碼與管理商品資料。

資料使用與分享

本 App 使用商品資料是為了在本機提供新增、編輯、刪除、搜尋、排序、月曆提醒、CSV 匯出、JSON 備份與還原功能。

本 App 不會販售、分享或轉移使用者建立的商品資料給第三方。

若使用者自行匯出 CSV 或 JSON 檔案，該檔案由使用者自行保存、傳送或刪除。

匯入、匯出與備份

CSV 匯出、JSON 備份與 JSON 還原都由使用者手動觸發。

JSON 備份檔包含商品資料與分類設定，請視為個人資料自行保存。

匯入檔案只會在本機讀取並合併到目前資料庫，不會上傳。

資料保存與刪除

本機資料會保留在使用者裝置中，直到使用者刪除商品、清除網站資料、移除 App，或刪除自行保存的備份檔。

使用者可在商品清單刪除個別商品，也可清除瀏覽器網站資料或移除 App 來刪除本機資料。

若曾匯出 JSON 或指定本機檔案位置，請另外刪除使用者自行保存的檔案。

資訊安全

本 App 以本機儲存為主要設計，開發者不會主動接收使用者建立的商品資料。

請使用者自行保管裝置、瀏覽器資料與匯出的備份檔，避免未授權的人取得資料。
