# CONTRIBUTING

## 開發原則
- 不破壞既有功能。
- 小步提交，單次只處理一類問題。
- 修改前先閱讀 `AGENTS.md`。

## 提交前自我檢查
- 是否影響以下場景：
  - 桌面表格模式（>=1526px）
  - 卡片模式（<1525px）
  - 深色主題
  - 月曆篩選後商品清單排版
- 是否有更新 `sw.js` 的 `CACHE_NAME`（若改動前端資產）。

## 程式風格
- 使用 UTF-8。
- 優先維持現有命名與檔案結構。
- 不引入不必要的新依賴。

## Issue 回報建議內容
- 裝置與瀏覽器版本
- 畫面寬度（或解析度）
- 重現步驟
- 預期結果 / 實際結果
