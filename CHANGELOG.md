# Changelog

所有重要變更皆記錄於此檔案。

## [v1.1.0] - 目前工作進度
- 版本與更新紀錄：
  - 新增 `version.js` 作為單一版本來源，集中管理目前版本、更新公告與歷史紀錄。
  - 目前語意化版本為 `v1.1.0`。
  - 設定頁新增「更新紀錄」區塊，顯示目前版本與更新內容。
  - 設定頁「關於」區塊右下角顯示目前版本號。
- PWA 圖示與快取：
  - 使用 `D:\AI Code\icons\icon21.png` 更新 PWA 圖示。
  - 新增 `icon-app-192.png`、`icon-app-512.png`，並更新 `favicon.ico`。
  - `manifest.webmanifest` 的 `theme_color` 同步改為淡紫色 `#8f82c7`。
  - 每次前端資產修改皆同步遞增 `sw.js` 的 `CACHE_NAME`。
  - 目前快取版本為 `expiry-manager-cache-v115`。
- 系統設定：
  - 手機版「返回」按鈕固定在與主頁「設定」按鈕相同的位置與尺寸。
  - 設定頁「返回」按鈕位置同步對齊主頁「設定」按鈕。
- 新增商品與版面：
  - 桌面左側新增商品區塊已移除。
  - 新增商品表單改為彈出式 modal。
  - 頂部導覽列保留「新增商品」與「設定」按鈕，並與「設定」按鈕同樣式。
  - 「新增商品」按鈕寬度加寬，填補移除月曆切換按鈕後的空間。
  - 小螢幕頂部標題列改為響應式排列，避免 320px 寬度位移，並加大標題與按鈕列的上下間隔。
- 商品效期月曆：
  - 新增商品效期月曆。
  - 月曆固定顯示。
  - 移除「關閉月曆 / 打開月曆」切換按鈕。
  - 月曆日期仍可篩選該日商品，再點一次取消。
- 商品清單與資料健康檢查：
  - 商品清單右側新增資料健康檢查篩選。
  - 顯示並可篩選：無日期、無條碼、重複條碼、已過期、60 天內即期、30 天內即期。
  - 「60 天內即期」健康檢查只計算 31 到 60 天內到期商品，避免與「30 天內即期」重複。
  - 健康檢查按鈕數量會依目前選擇的商品分類重新計算。
  - 點擊健康檢查項目可篩選商品清單。
  - 再點同一項或點頁面非互動區域，可取消健康檢查篩選並恢復預設清單。
  - 765px 以下健康檢查按鈕排列為 3 欄 x 2 排。
  - 手機卡片模式的全選框已固定在商品清單右側，與商品卡片 checkbox 對齊。
  - 排序欄中的「無日期」「無條碼」選項已移除，改由資料健康檢查篩選。
  - 修正商品列長按開啟條碼視窗時，瀏覽器右鍵選單仍會出現的問題；桌面滑鼠右鍵仍不觸發條碼視窗。
  - 補強卡片模式長按開啟條碼視窗後的原生右鍵選單攔截。
- 編輯商品：
  - 編輯商品視窗的有效日期欄位右側新增「選擇日期」按鈕，行為與新增商品一致。
- 鏡頭掃描與手電筒：
  - 掃描視窗的「關閉掃描」右側新增手電筒按鈕。
  - 手電筒按鈕狀態：`打開手電筒`、`長亮手電筒`、`關閉手電筒`。
  - 長亮模式下關閉掃描視窗時會保留 camera track，讓手電筒維持亮著。
  - 離開頁面時會強制關閉掃描與手電筒。
- 資料儲存：
  - 首次使用主頁時新增「選擇資料儲存方式」視窗。
  - 使用 IndexedDB 時會提醒：清除瀏覽器資料會刪除本專案資料。
  - iPhone / iPad Safari 備份提醒已加入首次視窗與設定頁「資料儲存」區塊。
  - 設定頁新增「資料儲存」區塊，顯示目前模式並可切換 IndexedDB / 本機檔案位置。
  - 「裝置資料檔」文案已統一改為「本機檔案位置」。
  - 主頁初始化時不再自動要求 File System Access 權限，只使用 `queryPermission()` 檢查，避免 `User activation is required` 錯誤。
  - 若本機檔案位置需要重新授權，會先顯示 IndexedDB 資料並提示使用者重新授權。
- 備份與提醒：
  - IndexedDB 模式下，每新增 30 筆商品會跳出 JSON 備份提醒。
  - 點「備份 JSON」會下載目前商品資料 JSON 並重置提醒計數。
  - 點「稍後」會關閉提醒並重置本輪計數。
- 重複條碼：
  - 新增商品或編輯商品時，若條碼已存在會跳出提醒。
  - 可選擇：覆蓋既有商品、仍然新增 / 仍然儲存、取消。
- 字體與 UI：
  - 移除所有自訂粗體樣式與 `<strong>` 標籤。
  - 覆蓋瀏覽器標題與表格標頭預設粗體，使用一般字重。
  - 修正 iOS Safari 備份提醒長文字在小寬度/橫向手機超出區塊問題。
- 專案管理文件：
  - 新增 `AGENTS.md`、`CONTRIBUTING.md`、`.editorconfig`。
  - 更新 `AGENTS.md` 桌面版面規則，符合目前新增商品 modal 與單欄月曆/商品清單布局。
